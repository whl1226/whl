# 将台堡·红歌新唱 - 数字声音博物馆

> 一个现代化的Web应用，用于欣赏经典红歌，了解革命历史，感受将台堡的文化底蕴。

## ✨ 功能特性

- 🎵 **红歌库** - 精选革命时期的经典歌曲
- 📜 **历史展示** - 详细的历史背景信息
- 🎯 **美观界面** - 现代化、响应式的用户界面
- 🖥️ **跨平台** - 支持Web和Tauri桌面应用

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev
```
在浏览器中打开 `http://localhost:5173`

### 构建生产版本
```bash
npm run build
```

### 预览构建结果
```bash
npm run preview
```

## 📁 项目结构

```
├── src/
│   ├── components/          # React 组件
│   │   ├── Navigation.tsx   # 导航栏
│   │   ├── Player.tsx       # 播放器
│   │   └── SongList.tsx     # 歌曲列表
│   ├── pages/               # 页面组件
│   ├── assets/              # 静态资源
│   ├── App.tsx              # 应用主组件
│   ├── App.css              # 应用样式
│   ├── index.css            # 全局样式
│   └── main.tsx             # 程序入口
├── index.html               # 主HTML文件
├── vite.config.ts          # Vite 配置
├── tsconfig.json           # TypeScript 配置
└── package.json            # 项目配置
```

## 🎨 设计特点

- 渐变蓝色主题，营造庄严的历史氛围
- 流畅的动画和过渡效果
- 玻璃态UI设计
- 完全响应式，适配各种屏幕尺寸

## 🔧 技术栈

- **React 18** - UI库
- **TypeScript** - 类型安全
- **Vite** - 快速构建工具
- **CSS3** - 样式
- **Tauri** - 桌面应用(计划中)

## 📖 使用指南

1. 选择一首歌曲
2. 查看歌曲信息和历史背景
3. 点击播放按钮（目前为UI演示，稍后支持真实音频）
4. 了解将台堡和红军文化
